OneDayMatch
概要

OneDayMatch は、「今日だけ誰かとつながりたい」「今日だけ誰かの投稿を見たい」「今日だけ少し話したい」 そんな気分の日に使う、1日限定型のマッチングWebアプリケーションです。

毎日やり取りを続けることや、関係が発展することを前提にせず、その日のみ誰かと出会える場を目指しています。

ユーザーは「今日だけ誰かと繋がりたい」として投稿した日に限り、同じ日に投稿しているの投稿を見ることができます。翌日になると投稿は表示されなくなり、関係も自然に区切られます。

サイトテーマ

「今日だけ、誰かとつながりたい人のための出会い」

テーマを選んだ理由

マッチングアプリは便利ですが、 「やり取りを続けなければならない」「関係を進める前提が疲れてしまう」 と感じてしまうことがあります。

一方で、

今日だけ誰かの投稿を眺めたい

今日だけ誰かと話したい

今日は深い関係までは求めていない

という日も確かにあります。

OneDayMatch は、そうした一時的な気持ちを否定せず、"今日だけ"という時間軸で人とつながれる場を作りたいと考え、このテーマを選びました。

制作のきっかけ

自分自身、マッチングアプリを使う中で、 「マッチした後のやり取りが負担になる」「関係を続ける前提で評価されているように感じる」 といった違和感を覚えることがありました。

その一方で、 「今日は少し誰かと話したい」「気が合いそうな人と軽くやり取りしたい」 と思う日もありました。

その日の気分で、無理なく出会えて、翌日にはきれいに終わる。 そんな場があってもいいのではないかと考えたことが、このアプリの出発点です。

解決したい課題

マッチ後の継続的なやり取りが負担になりやすい

出会いが「続くかどうか」で評価されがち

気軽に使いたいのに、心理的ハードルが高い

その日の気分に合った出会いがしづらい

課題に対する解決策

当日限定の投稿設計：投稿はその日限りで表示され、翌日には自動的に非表示

同日のユーザーのみ表示：今日出会いたい人同士だけが一覧に表示される

関係を残さない設計：継続を前提としないため、気軽にやり取りできる

投稿がない日も成立するUI：無理に誰かが出てくる設計にしない

ターゲットユーザー

マッチングアプリに疲れを感じている人

重い関係ではなく、軽い出会いを求めている人

その日の気分で出会いを選びたい人

今日だけ話せる相手を探している人

主な利用シーン

今日だけ誰かの投稿を見たいとき

今日だけ誰かと少し話したいと感じたとき

人とつながりたいけれど、重い関係は求めていない日

同じ日に投稿している人の投稿を閲覧

気になった投稿にリアクション（いいね・コメント）

補足

OneDayMatch は、出会いを「続けるかどうか」ではなく、 「今日マッチしたい」で考えるマッチングアプリです。

今日だけのやり取りでもいい。 何も起きない日があってもいい。

そんな前提で、気軽に使える出会いの場を目指しています。
